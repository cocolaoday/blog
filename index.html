關於我

了解我的背景、技術專長和生活興趣

我是李明華，畢業於國立台灣大學資訊工程學系，目前任職於一家科技新創擔任全端工程師。 在這五年多的職涯中，我參與了超過二十個大小不一的專案，從電商平台到 SaaS 工具， 積累了豐富的實戰經驗。

技術上，我熟悉 JavaScript 全端開發生態，前端使用 React、Next.js 打造高效能介面， 後端以 Node.js 搭配 PostgreSQL 構建穩健的 API 服務。 同時，我對 UI/UX 設計充滿熱情，深信「好的設計應該讓使用者感受不到設計本身」。

工作之餘，我喜歡用攝影記錄生活中的美好片刻，偶爾揹起登山包到山林中充電， 也保持每週閱讀的習慣，讓文字帶給我不同的視角。
